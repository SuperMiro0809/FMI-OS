# FMI-OS

Това хранилище съдържа решения на почти всички задачи за **Shell** и **C** от сборника по Операционни системи (ФМИ).

## Структура

- [`Shell/`](Shell/) — решения на задачите за Shell скриптове.
- [`C/`](C/) — решения на задачите за C.
- [`Exercises/`](Exercises/) — допълнителни упражнения.

Всяка папка съдържа собствен `README.md` с повече подробности за конкретните задачи и условията, при които са решени.
